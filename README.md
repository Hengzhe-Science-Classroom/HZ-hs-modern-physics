# Intro to Modern Physics (High School) · Interactive

**Relativity, quantum concepts, atomic models**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 2 · High School

## What You'll Learn

- Special relativity: time dilation, length contraction
- Photoelectric effect and photons
- Wave-particle duality
- Bohr model and atomic spectra
- Nuclear physics: fission and fusion
